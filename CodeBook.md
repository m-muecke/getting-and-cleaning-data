# Code Book

This code book gives a brief summary of the `tidy.txt` data set.

## Identifiers

- `subject` - The ID of the test subject.
- `activity` - The type of activity performed by the test subject.

## Activity Labels

- `1` (`WALKING`): Subject was walking.
- `2` (`WALKING_UPSTAIRS`): Subject was walking up a staircase.
- `3` (`WALKING_DOWNSTAIRS`): Subject was walking.
- `4` (`SITTING`): Subject was sitting.
- `5` (`STANDING`): Subject was standing.
- `6` (`LAYING`): Subject was laying down.
